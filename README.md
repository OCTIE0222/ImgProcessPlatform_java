**项目说明**
- 本项目是一个图像处理识别和展示的平台
- 从前端摄像头获取图像，一帧一帧传给后台服务器，中间通过activemq和redis做中间件，server端包含人脸识别和姿态估计两个工程（不完整）的各一份文件（参照）
- 局域网下基本实时传输图像，server处理完的图像，client能通过网页的形式访问。时延较少。
